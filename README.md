# Deadlight

Browser-based top-down co-op zombie survival with Hades-like combat feel and a
dark, claustrophobic atmosphere. 1–4 players, P2P (WebRTC), waves of zombies and
mini-bosses, loot, and an arsenal from pistol to chainsaw.

**Stack:** Phaser 3 · TypeScript · Vite · PeerJS (WebRTC)
